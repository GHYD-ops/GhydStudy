# Ghaid Study Project
**Owner:** Ghaid (GHYD-ops)
**Website:** https://ghaid-study.netlify.app/
Welcome to my personal project! This website is designed to simplify Calculus and Computer Science concepts for students through an interactive and modern user interface.
### 🛡️ Verification
This repository is the official source code for the educational website hosted at the link above, developed and managed by me, Ghaida.
## Tech Stack
The project is built using modern web standards to ensure speed, accessibility, and responsiveness:

HTML5: Semantic structure for better content organization.

CSS3: Modern design, including backdrop-filter and smooth CSS transitions for a premium user experience.

JavaScript (ES6+): Interactive features, including smart clipboard copying for payment details.

Typography: Using the Almarai font for a clean and professional Arabic reading experience.
